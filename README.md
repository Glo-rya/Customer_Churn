#Project Title

Predicting Customer Churn in a Telecommunications Company Using Machine Learning

##Overview 

This project develops  MACHINE LEARNING MODEL to predict when telecom customers are likely to churn. This is to help the company identify which costumers are risky and which ones to keep by using retention strattegies before they leave.

##Objectives

Predict customer churn using historical customer data.

Identify the most important factors associated with churn.

Build a model that can support proactive customer retention decisions.


##Dataset

The project uses the IBM Telco Customer Churn Dataset available from Kaggle or the IBM sample data repository.
[Telco-Customer-Churn.csv](https://github.com/user-attachments/files/29169933/Telco-Customer-Churn.csv)


##Technologies Used

Python , Jupyter Notebook, Pandas, Numpy, Matplotlib, Seaborn, Scikit-Learn

##Installations

pip install pandas numpy matplotlib seaborn scikit-learn

## Usage

Load the dataset into Jupyter Notebook.

Clean missing values and convert data types.

Perform exploratory data analysis.

Encode categorical variables.

Train a machine learning model.

##Findings

The analysis showed that churn is associated with variables such as tenure, monthly charges, and contract type. Customers with shorter tenure and higher monthly charges were more likely to churn.
